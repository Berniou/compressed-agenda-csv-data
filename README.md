# Compressed agenda csv data

## Description

This project is a set of fonctions. Each function generates 7x96 cells corresponding to a weekly timetable with a step of 15 minutes. Each day is splitted into 96 pieces i.e. 24 pieces (for 24 hours) and each piece is splitted into 4 inner pieces that gives 96 pieces.

Then for a week, we have 672 pieces.
